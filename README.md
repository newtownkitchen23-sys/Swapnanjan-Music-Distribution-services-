## 📌 Project Overview
The Emergency SOS App is a real-time safety application designed to provide immediate help during emergency situations. With a single tap, users can send an SOS alert along with their live location to trusted contacts or emergency services like the 112 helpline.

This application focuses on quick response, reliability, and user safety.

---

## 🎯 Key Features
- 🚨 One-tap SOS alert system  
- 📍 Real-time GPS location sharing  
- 👥 Emergency contact integration  
- 🔔 Instant alert notification  
- 📱 User-friendly interface  

---

## 🛠️ Technologies Used
- Android (Java/Kotlin)  
- Firebase  
- GPS Location Services  
- Google Maps API
